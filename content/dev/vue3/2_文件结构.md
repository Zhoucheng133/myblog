# 文件结构

`Vue3`的文件一般如下

## HTML内容

```html
<template>

</template>
```

注意，和`Vue2`不同，`template`下面可以有多个标签，不需要唯一的`div`标签

## script内容
```html
<script setup lang='ts'>

</script>
```

注意，如果你需要使用`Vue2`的语法，见`【Vue2】文件结构`，可以直接使用，并且两者可以同时存在（不推荐）

## style内容

```html
<style scoped>

</style>
```
和`Vue2`相同，添加上`scoped`可以防止样式穿透
