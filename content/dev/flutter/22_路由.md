# 路由

Flutter自带的`Router`（路由），可以在不同页面之间转换

## 跳转到某个页面

```dart
// 如果是在函数中使用，传递BuildContext参数
void route(BuildContext context){
  Navigator.push(
    context, 
    // Material风格的路由跳转
    MaterialPageRoute(builder: (context) => newPage())
    // iOS风格的路由跳转
    CupertinoPageRoute(builder: (context) => newPage())
  );
}
```

## 返回页面

```dart
// 如果是在函数中使用，传递BuildContext参数
void route(BuildContext context){
  // 也可以用于弹窗消失
  Navigator.of(context).pop();
}
```
