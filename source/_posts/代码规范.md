# 代码规范

## 缩进

尽量不要使用`制表符`

`Viual Studio Code`的默认缩进是4个空格，对于需要切换到2个空格的时候，自定义工作区的缩进：
```json
{
  "editor.tabSize": 2
}
```

### 使用四个空格作为缩进的

- `C/C++`
- `Java`
- `Python`
- `Go*`

### 使用两个空格作为缩进的

- `JavaScript`
- `Dart`
- `Ruby`
- `Vue`

## 末尾分号

### 必须添加

- `C/C++`
- `Java`
- `Dart`

### 一般需要添加，但是不添加也可以

- `JavaScript`

### 不需要添加

- `Python`
- `Go`
- `Ruby`

## 大括号

针对需要添加大括号来实现语法的语言，一般这样写

```js
function fun(){
  // 函数内容
}
```

## (对于`JavaScript`)函数与箭头函数

一般来说可以使用箭头函数，不使用普通函数

```js
aixos.get("/user")
.then((response)=>{
  // 函数内容
})
```
